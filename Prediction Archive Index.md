## Purpose

The purpose of this index is to organize the prediction records derived from the **Information Existence Hypothesis (IEH)**.

Each prediction record is intended to be:

- dated
- traceable
- falsifiable or partially falsifiable
- linked to specific IEH corollaries
- open to future evidence updates

This index helps distinguish between the theoretical framework of IEH and its empirical prediction archive.

## Status Categories

Prediction records may use the following status labels:

| Status | Meaning |
|---|---|
| Active | The prediction is currently being tracked. |
| Supported | The prediction has received meaningful supporting evidence. |
| Partially Supported | Some aspects of the prediction have received support, while others remain uncertain. |
| Unresolved | There is not yet enough evidence to evaluate the prediction. |
| Falsified | The prediction has been clearly contradicted by future evidence. |
| Revised | The prediction has been modified or narrowed in response to new evidence. |
| Archived | The record is preserved for historical reference but is no longer actively tracked. |

## Recommended Prediction Record Format

Each prediction record should ideally include:

```text
Title
Prediction ID
Date Created
Date Updated
Related IEH Corollaries
Prediction Statement
Falsification Conditions
Evidence Log
Current Assessment
Revision History
```

This format is intended to make each prediction more transparent, testable, and historically traceable.
