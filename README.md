## Purpose

The purpose of this repository is to record, track, and revise specific predictions derived from the Information Existence Hypothesis.

Each prediction record is intended to be:

- traceable
- dated
- falsifiable or partially falsifiable
- linked to specific IEH corollaries
- open to future evidence updates

This archive is not a replacement for the main IEH theory repository.  
It functions as the empirical and observational companion layer of IEH.
